Program Overview

The MSIT program is designed to equip students with the skills and knowledge needed to excel in the field of information technology. Our curriculum is tailored to meet the needs of both beginners and experienced professionals.

Curriculum

Data Structures and Algorithms

Learn the foundational concepts of data structures and alogorithms to build efficient systems.

Database Management Systems

Understand how to design and manage databases using SQL and NoSQL technologies.

Web Development

Explore front-end and back-end technologies to build dynamic, full-stack web applications.

Cloud Computing

Discover cloud virtualizatin,and containerization to deploy scalable applications.

Cybersecurity

Learn the essientials of protecting systems and data from the malicious attacks and breaches.

Machine Learning

Gain knowledge in ML algorithms and how to apply them to real-world problems

Admission Requirements

To apply for the MSIT program, you will need:

  • A bachelor's degree in a related field
  • Official transcripts
  • Resume/CV
  • Statement of purpose
  • Two letters of recommendation
